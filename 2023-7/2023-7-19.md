# 1.admin-template

​	1.1 搭建 vue 后台

​		搭建环境 node -v 10.24.1 ---------

​		npm -v 6.14.12 -----------------------

​		node-sass -v 4.0 ---------------------

​	1.2 框架改造流程

​		第一步：在 src 目录下的 router/index.js 中添加路由

​		第二步：设置跳转页面路径，跳转页面的路径设置为 views 目录下的 vue 文件

​		第三步：在api文件夹创建 js 文件，定义接口路径

​		第四步：在页面引入 js 文件，使用 axios 进行接口调用，把接口返回数据在页面显示
